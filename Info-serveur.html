Arène de combat

 le royaume possède un salon arène, ce sera ici que vous allez pouvoir trouver un confrère pour jouer, ou plusieurs. Six roles s'offrent à vous, Smash Bros U., Mario Kart, Mario Party, Splatoon, Minecraft, Pokemon. Vous pouvez ping le rôle de votre choix pour trouver un Toad.  N'hésitez pas à ping, le rôle est fait pour ça et les Toads choissisent de l'avoir donc d'être notifié. 

Vocaux

durant vos matchs, vous avez la possibilité d'aller en vocal dans plusieurs salons : 1 vs 1, 2 vs 2, arène (8 places). A vous de choisir quel vocal est le mieux adapté.  NB : la particiption en vocal est comptabilisée dans le classement du serveur et même plus rentable que d'écrire.

  CommandesRoyaume !

Sur ce serveur, tu pourras intéragir avec Toad, le bot du serveur. Avec les commandes slash, tu pourras lui demander ton niveau, tes sanctions et ta date de bienvenue. Tu pourras également le solicité avec les commandes images pour transmettre tes emotions.

IEt attention, le bot peut se montrer un peu sensible, donc attention à ne pas trop l'énervé...

  Toadhub & DiscussionRoyaume !

Le salon discussion est le salon principal du serveur, les Toads peuvent discuter entre eux en messages ou aller dans le salon pour parler en vocal. Mais, si les Toads veulent parler d'un sujet en particulier, il est possible de créer un salon dans le Toadhub pour créer un endroit spécialement pour un type de sujet ! 

Très pratique pour ne pas être dérangé lors de vos discussions !
