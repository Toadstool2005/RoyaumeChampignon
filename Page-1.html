► Les commandes du bot 

Nous avons un bot spécialement conçu pour le Royaume Champignon ! Celui-ci va gérer la modération, l'économie, l'expérience, le ladder, les vocaux ainsi que de nombreuses commande slash !  Il sera là tout au long de votre séjour et dès la première seconde de votre arrivé. Et il peut même vous contacter en mp !

► Modération  

En cas d'avertissement : générale
- 1 ou 2 infraction(s) => Aucune infraction
3 infractions => Mute un jour.
4 infractions => Mute 1 semaine
5 Infraction => ​Bannissement

En cas d'avertissement : Tournoi
- 1 ou 2 infractions : Pas de sanction
-3 infractions : Bannissement des évènements et infraction générale. Le nombre d'avertissement retourne à 0 pour le Tournoi.

Sanction d'un Ban :
-Mention Everyone, Here, Toad
- Contenu NSFW
- Toutes formes de promotions

En cas d'infraction, un membre du staff via le bot vous infligera un avertissement. Dans de rares cas, des sanctions, cela dépend de votre infraction. ​Nous avons 2 sortes d'infractions : tournoi et général. Tournoi, c'est durant les événements et le reste général. Le premier et deuxième avertissement, vous n'aurez pas grand-chose, mais ensuite, c'est là que le bot vous sanctionnera. Vous allez toujours être averti lors d'une infraction, le bot vous enverra la raison, la date et le membre qui vous a averti. Et bien évidemment, vous pouvez vous exprimer si votre infraction vous semble injuste.

Pour éviter tout mal-entendu, suffit de lire le règlement !

► Le système monétaire 

Nous avons un système qui vous permet de gagner des pièces, il y en a 3 types :
étoile : Obtenable durant les tournois/CB. Sert à classer les Toads pour savoir qui est le meilleur ! C'est le champi Classement. 
Super Pièce : Obtenable avec le ladder
Pièce : Obtenable avec la commande /pièce quotidienne ou avec /boite mystère. Sert à acheter des items dans le shop.

Les 3 types sont donc bien différentes, 2 types de pièces servent plus comme score pour le ladder et le classement et ne peuvent être échangé ou donné.

► Votre profil personnalisé 
Vous pouvez vous créer un profil pour indiquer certaines informations. Ceci remplace les rôles personnages et le salon code ami. Ce profil sera important pour les tournois, il indique vos mains et votre Team. Ce n'est rien qu'a vous donc profitez s'en ! Vous pouvez ajouter ceci : 
- Main : (obligatoire pour afficher votre profil)
_Ajouter un second (Vous pouvez en ajouter 3 seulement pour les @👑 Toad fidéle )
_Ajouter une note (seulement pour @👑 Toad fidéle )
_Ajouter votre code ami
_Ajoutez votre Team. (Il n'y a pas tout mais vous pouvez demander de l'ajouter)
-Ajouter votre localisation
Pour ajouter ces élements, il vous suffit de faire /ajoute puis sélectionner ce que vous souhaitez ajouter ! Pour avoir votre profil : /profil, vous pouvez le mettre en privé ou en public !
